## ğŸ”„ The Lifecycle of a DevOps Task
Technical work doesn't appear out of thin air. It follows a structured breakdown:

1.  **The Epic**: A large, high-level objective (e.g., "Enable Global Payments").
2.  **The User Story**: A smaller, functional requirement (e.g., "As a user, I want to pay via UPI").
3.  **The Task**: The actual technical work assigned to you (e.g., *"Provision AWS RDS instance for Payment Service"*).
